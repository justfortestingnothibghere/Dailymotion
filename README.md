## 📌 Overview

**Dailymotion M3U8 Scraper** is a powerful Python script that fetches **direct M3U8 stream URLs** from Dailymotion videos using their internal metadata API.

Perfect for:
- Streaming tools 🎥
- Downloaders 📥
- Telegram bots 🤖
- Reverse engineering projects 🧠

---

## ✨ Features

- ✅ Extract M3U8 stream link
- ✅ Supports both URL & Video ID
- ✅ Async (aiohttp based)
- ✅ Retry mechanism for reliability
- ✅ Clean & structured response
- ✅ Lightweight & fast
